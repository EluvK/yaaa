# yaaa

Yet Another AI Assistant. A toy project while learning Flutter.

## Supported Features

- ✅ Multi-Platform Support (Android, Windows, ...)
- ✅ Multi-LLM Support
- ✅ Customizable Prompt
- ✅ Continuous Conversation
- ✅ Markdown and Latex Support
- ✅ Dark Mode
- ✅ Keyboard Shortcuts
- ✅ Multi-Language Support
